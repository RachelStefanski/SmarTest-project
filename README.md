# SmarTest-project
Automated test checking

Project Description:
Online testing site that allows for automatic answer checking.
The site is primarily intended for internal tests, where the scope of knowledge in the answers depends on the scope of the material taught in class.
This means that an answer that is labeled as correct does not necessarily contain all the knowledge that should be in the real answer in the world.
For example, a teacher who taught only 3 reasons for a phenomenon when there are 5 reasons will mark an answer of 3 reasons as correct even though in the real world it is missing/incorrect.
On the website, the teacher will be able to create friendly, easy-to-use questionnaires that are customized to the needs of the class.

Types of possible questions:
Multiple choice questions - students must choose the correct answer from a number of given options.
True/False questions - students must determine whether the statement is true or false.
Open questions - a free answer is required, in which the student is asked to formulate a complete answer to the question or problem.

Types of possible answers:
Choice answers - answers to American or True/False questions.
Detailed answers - answers that provide a more in-depth explanation, usually in open questions, including examples, analyses, and extensive explanations.

The questions vary depending on the topic and level of the test.

In order to allow Automatic checking of answers while saving time for the teacher, who must answer a sample questionnaire at the level of knowledge expected from the students so that it serves as a model for testing the tests.
The site will analyze the teacher's and student's answers, compare them according to varying criteria, and provide advanced results analysis that allows for understanding of the students' performance.
Students will be able to easily access the questionnaires, a friendly interface for answering, and receive immediate grades that include analysis of results that helps them improve their understanding and performance.

Input Output Settings:
Input:
Create an exam: questions, types of questions, submission date, specified time for the test, and special instructions.
Students' answers to the questions, including choices in multiple-choice and open-ended tests.
Output:
Scores after answering the questionnaire, including error analysis.
